# 高中生物实践作业解题思路：设计或制作性状分离模拟器

## 1. 审题

题目要求设计或制作一个“性状分离模拟器”，模拟杂交水稻申优 28 的自交后代中胚乳淀粉类型相关性状的分离情况，并完成设计报告。

题干给出的遗传信息如下：

- 控制胚乳淀粉类型的是一对等位基因：`W` 和 `w`。
- `W`：直链淀粉含量正常，表现为非糯性，显性。
- `w`：直链淀粉含量低，表现为糯性，隐性。
- 亲本申优 28 的基因型为 `Ww`。
- 问题是：`Ww` 自交后代会出现哪些基因型和表型，比例是多少。
- 报告要求包括：
  - 模拟器的设计说明。
  - 模拟器的使用说明。
  - 实验记录表。
  - 分析与讨论。
- 若做到 L2 层级，需要用模拟器完成至少 50 次实验，并汇总统计结果。

因此，本题本质上是模拟一次经典的孟德尔一对相对性状杂合子自交：

```text
P: Ww x Ww
```

其中两个亲本产生配子的概率相同：

```text
W 配子概率 = 1/2
w 配子概率 = 1/2
```

## 2. 理论预测

用棋盘格分析 `Ww x Ww`：

| 配子 | W | w |
| --- | --- | --- |
| W | WW | Ww |
| w | Ww | ww |

理论基因型及比例：

| 基因型 | 概率 | 比例 |
| --- | --- | --- |
| WW | 1/4 | 25% |
| Ww | 1/2 | 50% |
| ww | 1/4 | 25% |

理论表型及比例：

| 表型 | 对应基因型 | 概率 | 比例 |
| --- | --- | --- | --- |
| 非糯性 | WW、Ww | 3/4 | 75% |
| 糯性 | ww | 1/4 | 25% |

结论：

```text
自交后代理论基因型比例为 WW : Ww : ww = 1 : 2 : 1。
自交后代理论表型比例为 非糯性 : 糯性 = 3 : 1。
```

## 3. 模拟器设计方案

本题可以设计成实物模拟器，也可以设计成计算机程序。为了方便统计 50 次以上实验，建议使用 Python 编写一个简单模拟器。

### 3.1 模拟器代表含义

将一个亲本 `Ww` 的减数分裂过程抽象为“随机产生一个配子”：

- 随机取到 `W`，表示该亲本产生含 `W` 基因的配子。
- 随机取到 `w`，表示该亲本产生含 `w` 基因的配子。
- 两个亲本各随机产生一个配子。
- 两个配子结合，形成一个后代基因型。
- 根据基因型判断表型：
  - 只要含有 `W`，即 `WW` 或 `Ww`，表型为非糯性。
  - 只有 `ww`，表型为糯性。

### 3.2 Python 核心逻辑

核心算法可以写成：

```python
import random

def make_gamete():
    return random.choice(["W", "w"])

def make_offspring():
    gamete1 = make_gamete()
    gamete2 = make_gamete()
    genotype = "".join(sorted([gamete1, gamete2], key=lambda x: 0 if x == "W" else 1))
    phenotype = "糯性" if genotype == "ww" else "非糯性"
    return gamete1, gamete2, genotype, phenotype
```

完整程序思路：

1. 设定实验次数，例如 `n = 50`。
2. 重复进行 `n` 次模拟受精。
3. 每次记录：
   - 实验序号。
   - 亲本甲产生的配子。
   - 亲本乙产生的配子。
   - 后代基因型。
   - 后代表型。
4. 统计 `WW`、`Ww`、`ww` 的个数。
5. 统计非糯性和糯性的个数。
6. 计算实际比例，与理论比例 `1:2:1` 和 `3:1` 比较。

### 3.3 模拟器设计说明可写内容

本模拟器用计算机程序模拟 `Ww` 自交过程。程序中用列表 `["W", "w"]` 表示每个杂合亲本能够产生的两种配子。每次实验时，程序分别从两个亲本的配子列表中随机抽取一个配子，模拟减数分裂产生配子和随机受精过程。两个配子结合后得到后代基因型，再根据显隐性关系判断表型。重复实验至少 50 次后，程序统计各种基因型和表型出现次数，比较实际统计比例与理论预测比例。

## 4. 模拟器使用说明

使用步骤：

1. 打开 Python 程序。
2. 输入或设定实验次数，建议不少于 50 次。
3. 运行程序。
4. 程序每次随机产生两个配子，并输出后代基因型和表型。
5. 将每次结果填入实验记录表。
6. 实验结束后，汇总 `WW`、`Ww`、`ww` 的数量和比例。
7. 汇总非糯性、糯性的数量和比例。
8. 将实际结果与理论结果比较，分析偏差原因。

注意事项：

- 实验次数越多，实际比例越接近理论比例。
- 每次实验必须独立随机，不能人为调整结果。
- 若只做 50 次，结果可能会与理论比例有一定差异，这是随机误差导致的正常现象。

## 6. 分析与讨论

理论预测：

```text
Ww 自交后代基因型比例为 WW : Ww : ww = 1 : 2 : 1。
表型比例为 非糯性 : 糯性 = 3 : 1。
```

实际模拟结果可能与理论比例不完全相同。若只进行 50 次实验，随机抽样次数较少，偶然性比较明显，因此实际比例可能偏离理论比例。随着实验次数增加，例如增加到 200 次、500 次或 1000 次，实际统计结果一般会逐渐接近理论比例。

若实际结果与理论结果存在偏差，可从以下方面分析：

1. 随机误差：实验次数有限，每次配子组合具有偶然性。
2. 样本量不足：50 次实验数量较少，统计结果不一定正好等于理论值。
3. 操作或记录错误：如果人工记录，可能出现漏记、错记或统计错误。
4. 程序设置问题：如果程序没有保证 `W` 和 `w` 出现概率均为 `1/2`，会影响模拟结果。

综合结论：

本模拟器能够模拟杂合子 `Ww` 自交时等位基因分离和随机受精的过程。模拟结果可以帮助理解孟德尔分离定律：杂合子在形成配子时，成对的等位基因彼此分离，分别进入不同配子；受精时雌雄配子随机结合，从而产生一定比例的后代基因型和表型。

## 7. 可提交报告的简短结论

申优 28 的基因型为 `Ww`，自交时两个亲本都能产生 `W` 和 `w` 两种配子，比例均为 `1:1`。随机受精后，后代可能出现 `WW`、`Ww`、`ww` 三种基因型，理论比例为 `1:2:1`。由于 `W` 对 `w` 为显性，所以 `WW` 和 `Ww` 表现为非糯性，`ww` 表现为糯性，理论表型比例为非糯性：糯性 = `3:1`。模拟实验中，实际结果可能因随机误差与理论比例略有差异，但实验次数越多，统计结果越接近理论比例。

## 8. 扩展题：加入第二对基因 Y/y

在原题基础上，可以再增加一对控制粒形的等位基因 `Y` 和 `y`，把亲本基因型扩展为：

```text
P: WwYy x WwYy
```

设定两对基因分别控制两种性状：

- `W/w` 控制胚乳淀粉类型：
  - `W_`：非糯性。
  - `ww`：糯性。
- `Y/y` 控制粒形：
  - `Y_`：长粒。
  - `yy`：短粒。

其中 `_` 表示该位置可以是显性基因，也可以是隐性基因。例如 `W_` 包括 `WW` 和 `Ww`，`Y_` 包括 `YY` 和 `Yy`。

### 8.1 配子类型

亲本 `WwYy` 在形成配子时，两对等位基因独立分离、自由组合，可形成 4 种配子：

| 配子类型 | 理论概率 |
| --- | ---: |
| WY | 1/4 |
| Wy | 1/4 |
| wY | 1/4 |
| wy | 1/4 |

### 8.2 16 种组合

`WwYy x WwYy` 可用 4 x 4 棋盘格表示：

| 配子 | WY | Wy | wY | wy |
| --- | --- | --- | --- | --- |
| WY | WWYY | WWYy | WwYY | WwYy |
| Wy | WWYy | WWyy | WwYy | Wwyy |
| wY | WwYY | WwYy | wwYY | wwYy |
| wy | WwYy | Wwyy | wwYy | wwyy |

这 16 种组合不是 16 种不同基因型，而是 16 个等概率的受精结果。其中有些基因型会重复出现。

### 8.3 两个性状组合的理论结果

根据显隐性关系判断：

| 性状组合 | 对应基因型 | 出现份数 | 理论比例 |
| --- | --- | ---: | ---: |
| 非糯性、长粒 | `W_Y_` | 9 | 9/16 |
| 非糯性、短粒 | `W_yy` | 3 | 3/16 |
| 糯性、长粒 | `wwY_` | 3 | 3/16 |
| 糯性、短粒 | `wwyy` | 1 | 1/16 |

因此，若 `W/w` 和 `Y/y` 两对基因独立遗传，且分别控制糯性和粒形，则自交后代两个性状组合的理论比例为：

```text
非糯性长粒 : 非糯性短粒 : 糯性长粒 : 糯性短粒 = 9 : 3 : 3 : 1
```

如果只看糯性这一种性状，不管长粒还是短粒，那么：

```text
非糯性 = 9/16 + 3/16 = 12/16
糯性 = 3/16 + 1/16 = 4/16
非糯性 : 糯性 = 12 : 4 = 3 : 1
```

如果只看粒形这一种性状，不管非糯性还是糯性，那么：

```text
长粒 = 9/16 + 3/16 = 12/16
短粒 = 3/16 + 1/16 = 4/16
长粒 : 短粒 = 12 : 4 = 3 : 1
```

### 8.4 扩展版 Python 模拟器思路

扩展版模拟器可以把每个亲本产生的配子从原来的 `["W", "w"]` 改成：

```python
gametes = ["WY", "Wy", "wY", "wy"]
```

每次实验时：

1. 从亲本甲的 4 种配子中随机抽取 1 个。
2. 从亲本乙的 4 种配子中随机抽取 1 个。
3. 合并两个配子，得到后代的两对基因。
4. 根据 `W/w` 判断糯性或非糯性。
5. 根据 `Y/y` 判断长粒或短粒。
6. 统计 4 种性状组合出现次数。

核心代码示例：

```python
import random

def make_gamete():
    return random.choice(["WY", "Wy", "wY", "wy"])

def make_offspring():
    gamete1 = make_gamete()
    gamete2 = make_gamete()

    w_genes = sorted([gamete1[0], gamete2[0]], key=lambda x: 0 if x == "W" else 1)
    y_genes = sorted([gamete1[1], gamete2[1]], key=lambda x: 0 if x == "Y" else 1)

    genotype = "".join(w_genes) + "".join(y_genes)

    starch_trait = "糯性" if w_genes == ["w", "w"] else "非糯性"
    grain_trait = "短粒" if y_genes == ["y", "y"] else "长粒"
    phenotype = starch_trait + grain_trait

    return gamete1, gamete2, genotype, phenotype
```

扩展版实验记录表可写为：

| 实验序号 | 亲本甲配子 | 亲本乙配子 | 后代基因型 | 胚乳性状 | 粒形 | 性状组合 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 |  |  |  |  |  |  |
| 2 |  |  |  |  |  |  |
| 3 |  |  |  |  |  |  |
| ... |  |  |  |  |  |  |
| 50 |  |  |  |  |  |  |

扩展版汇总统计表：

| 性状组合 | 数量 | 实际比例 | 理论比例 |
| --- | ---: | ---: | ---: |
| 非糯性长粒 |  |  | 9/16 |
| 非糯性短粒 |  |  | 3/16 |
| 糯性长粒 |  |  | 3/16 |
| 糯性短粒 |  |  | 1/16 |

扩展版结论可以写成：

`WwYy` 自交时，亲本可产生 `WY`、`Wy`、`wY`、`wy` 四种配子，比例为 `1:1:1:1`。随机受精后，如果 `W/w` 控制糯性、`Y/y` 控制粒形，且两对基因独立遗传，则后代出现四种性状组合：非糯性长粒、非糯性短粒、糯性长粒、糯性短粒，理论比例为 `9:3:3:1`。
